# Ki-Station

Currently under design phase. UI is designed on [Penpot](https://design.penpot.app). 
Development will start soon and it will be built with C# and Avalonia UI.

# About the software
The Software will be a control station widget, like those linux KDE systems have.
In my mind my vision for it is to contain the most usefull things a user will need on a single panel.
So it will have features like an intergrated Bash Console, Media Control, time and weather info, KDE connect and more...

Along with that i'll try to add as much customization as possible with options like:
  - Color Picking for most UI Components,
  - Chnage the Buttons for other software or system functions the user might want to add.
  - Size Control
with the goal to make the regular user avoid having to access the code in order to so.

The code will be FOSS so everyone will be able to Access it, Watch it and Fork it to fit it on its own parameters!
